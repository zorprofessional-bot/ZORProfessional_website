Now build a simple admin panel for the ZOR Professional website.

Important:
The public website is a no-scroll deck-style presentation interface.

Do not create a classic CMS for long scrolling pages.
Do not bring back the old page_sections model.

The admin must manage:

* deck chapters
* deck slides
* products
* blog posts
* leads
* career positions
* career applications
* site settings

Follow:

* PROJECT_BRIEF.md
* DESIGN_SYSTEM.md
* CONTENT_STRUCTURE.md
* DECK_ARCHITECTURE.md

The public website concept:

* main navigation items are chapters
* each chapter contains multiple slides
* public pages use DeckShell, DeckPage, DeckSlide, DeckControls and SlideIndicator
* admin pages may use normal scrolling
* blog article pages may have internal scrollable reader panels
* public deck pages must remain no-scroll

Admin route:

* /admin

Use Supabase Auth.

Support multiple users with roles:

* admin
* editor
* viewer

Role rules:

* viewer can only read admin content
* editor can manage content, products, blog, leads and career content
* admin can manage everything, including site settings and user-related admin configuration if implemented

If full user management is too much for now, document how to create users in Supabase manually.

Admin layout:

* simple and practical
* sidebar on desktop
* simple top navigation or drawer on mobile
* clean forms
* clean tables
* filters where useful
* status badges
* save/cancel buttons
* no unnecessary charts
* no enterprise dashboard complexity

Important slide editing rules:

* show a helper note:
  “Javni web je prezentacija bez skrolanja. Svaki slajd treba imati jednu jasnu poruku i kratak tekst. Za duži tekst koristite blog.”
* show character guidance:
  title: recommended under 70 characters
  body: recommended under 220 characters
* do not technically block longer text unless easy, but warn visually

Slide preview:
Create a simple preview area that shows roughly how the slide will look in the public deck.
It does not need to be perfect, but it should help avoid bad long text.
